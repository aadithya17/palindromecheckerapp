# Palindrome Checker App

This is a simple Java console application that checks whether a given word or number is a palindrome.

A palindrome is a word, phrase, or number that reads the same forward and backward.

Examples:
madam → Palindrome  
racecar → Palindrome  
hello → Not a Palindrome

## Features

- Takes input from the user
- Checks if the input is a palindrome
- Works for both words and numbers
- Displays the result in the console

## Technologies Used

- Java
- Scanner class for user input
- String manipulation

## How the Program Works

1. The user enters a word or number.
2. The program reverses the input.
3. The reversed value is compared with the original input.
4. If both are the same, it is a palindrome. Otherwise, it is not.

## How to Run the Program

1. Compile the program

javac PalindromeChecker.java

2. Run the program

java PalindromeChecker

## Example Output

Enter a word or number:
madam

madam is a Palindrome

Enter a word or number:
hello

hello is not a Palindrome
